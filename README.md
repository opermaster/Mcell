# Mcell
Excel-like program that is made for interacting with table and applying for cells a primitive mathematical expressions.

#### Available operations
1. "+"
2. "-"
3. "/"
4. "*"
#### Avaliable mathematical functions
1. Abs
2. Sin
3. Cos
4. Sqrt
#### Example of 
```console
1+cos1*2
10+2+sqrt25
```
